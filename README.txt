== Description ==
WordPress Email Base Logo Change plugin is easy to use plugin. It gives you the power to convert the boring WordPress login page logo with personalized login logo from your WordPress Avatar. This plugin helps to improve your brand authority and also helps to improve user experience and you can change wordpress default logo and display your WordPress Avatar. It provides you nice user interface while doing login or register via WordPress backend on your site.

= What is used for this plugin? =

WordPress Email Base Logo Change plugin is for these who like to update WordPress admin logo based on their email address.

= Plugin Functionality: = 

* Easy to use.
* Provide nice user experience.
* Change WordPress logo based on their email address.
* Change WordPress logo and use WordPress Avatar.

== Installation ==

* Download the plugin
* Upload the folder "wp-email-base-logo-change" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!
* You can access the setting page from here Settings -> General.
* Enable WP Email Base Logo Change form here Settings -> General.
